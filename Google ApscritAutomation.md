Google Apps Script를 활용한 자동화 입문

매일 반복되는 데이터 옮기기, 설문 응답 확인 후 메일 보내기 등은 이제 자동화로 가능합니다.
별도의 설치 없이 브라우저에서 즉시 실행 가능하다는 점이 가장 큰 장점입니다.
 
Apps Script 활용 단계입니다
1. 코드 작성요청 : 제미나이, 챗지피티, 클로드 등에서 google apps script 코드 요청
2. Google Apps Script 실행 : 드라이브 내에서 직접 스크립트를 생성해 환경 구축 
3. 권한 승인 : 데이터 보안을 위한 안전한 접근 허용으로 설정 ->고급으로 들어가야함
4. URL 실시간 확인 : 생성된 폼의 URL을 즉시 추출하여 실시간 공유 체계 마련
5. 트리거 설정 :설문 제출 즉시 담당자 알림 및 데이터 기록 자동화

주의점 : 프롬프트 요청시, “매서드가 최근 변경되었을 수도 있으므로, 구글 공식문서 조사를 먼저 해줘” 추가
