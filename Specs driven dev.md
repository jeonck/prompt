쓰는 방법은 간단해요.

설치 한 줄.
`npx skills add addyosmani/agent-skills`

Claude Code에서는:
`claude plugin add agent-skills`

설치하면 슬래시 커맨드 7개가 생겨요.
`/spec` — 스펙 먼저 쓰기.
`/plan` — 태스크 분해.
`/build` — 점진적 구현.
`/test` — TDD 워크플로.
`/review` — 코드 리뷰.
`/code-simplify` — 코드 단순화.
`/ship` — 배포 전 체크리스트.

에이전트 페르소나도 3개 있습니다.
code-reviewer, test-engineer, security-auditor.
각각 코드 리뷰, 테스트 전략, 보안 점검에 특화돼 있어요.￼
