[시계열 예측을 쉽게, Google TimesFM]

TimesFM은 Google Research가 개발한 시계열 예측용 대규모 사전학습 모델입니다. 어제 GitHub Trending에 다시 올라와서, 이번에 직접 데이터로 테스트해봤는데 꽤 흥미로운 결과를 빠르게 받아볼 수 있었습니다. 

시계열 예측의 진입장벽이 점점 낮아지고 있는 흐름이 보입니다.

---

TimesFM은 2023년 논문 공개 이후 꾸준히 발전해온 시계열 예측 Foundation Model입니다. 2025년에 v2.5로 업데이트되면서 200M 파라미터로 경량화되었고, 최근 에이전트 지원과 외부 변수(XReg) 기능이 추가되면서 다시 주목받고 있는 것 같습니다.

100B 시계열 데이터포인트로 사전학습되어 있어, 튜닝 없이 데이터만 넣으면 바로 예측합니다. 오픈소스 + BigQuery 공식 지원도 합니다. Apache 2.0 라이선스, Hugging Face에서 다운로드 가능합니다.

🧪 직접 테스트해봤습니다.
- 벤처투자 데이터 82개월치(2017~2023, 약 1만 건)를 넣어봤습니다.
- GPU 없는 일반 Windows PC에서 코드 몇 줄로 2024년 월별 예측 성공
- 매년 12월 투자 급증 패턴을 모델이 스스로 포착한 점이 인상적

즉, 별도 튜닝 없이 빠른 속도로 시계열 예측 결과가 나왔습니다.

다만 일부 커뮤니티 반응은 "가능성은 인정, 프로덕션은 아직"이라는 의견도 있습니다. 고빈도(분/시간 단위) 데이터에서는 성능이 하락한다는 실무 테스트 사례와, 단변량 중심이라 외부 변수 반영이 제한적이라는 지적도 있고요. Amazon Chronos-2 등 경쟁 모델과 벤치마크 1위를 주고받는 상황이기도 합니다.

그러나 개인적으로 TimesFM의 가장 큰 의미는 "정확도 혁신"보다 "접근성 확대"에 있는 것 같습니다. TimesFM은 "데이터만 있으면 일단 시도해볼 수 있다"는 다음 단계를 보여주고 있습니다. 아직 만능은 아니지만, 월별 데이터 같은 탐색적 분석에서는 충분히 유용했습니다.
