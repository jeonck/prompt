또다른 MiroFish 대박 프로젝트입니다. 트레이더 Op0jogggg는 NBA 데이터로 스포츠 예측 베팅으로 20억 원을 벌었습니다. 2025년 11월 Polymarket에 가입했습니다. 4개월 만에 누적 수익 148만 달러(약 20억 원). 현재 보유 포지션 180만 달러.

더 좋은 모델을 만든 것이 아닙니다. 더 좋은 군중을 시뮬레이션한 것입니다.

그가 사용한 도구는 MiroFish입니다. 최근 GitHub 글로벌 트렌딩 1위를 찍으며 화제가 된 오픈소스 스웜 인텔리전스 엔진입니다. MiroFish 기반 수익 사례가 연달아 보고되고 있는데, 이번이 가장 큰 규모입니다.

Op0jogggg의 방식은 이렇습니다.

→ 3시즌 분량의 선수 스탯, 팀 폼, 매치업 이력, 부상 확률, 라인 무브먼트를 입력
→ MiroFish가 4,096개의 AI 에이전트(애널리스트, 베터, 오즈메이커 등)를 생성
→ 에이전트들이 서로 토론하고, 클러스터를 형성하고, 의견을 수렴
→ 합의를 16,695건의 예측 이력으로 훈련된 트랜스포머에 입력
→ Polymarket 실시간 배당과의 차이가 켈리 기준을 초과하면 진입

레이커스 배당 40센트. MiroFish 예측 62%.
한 포지션에 19만 달러를 넣었습니다.

✅ 핵심은 데이터가 아닙니다. 모든 샤프 베터에게 데이터는 있습니다
✅ 차이는 4,096명의 가상 인간이 데이터를 처리하는 방식을 시뮬레이션한 것입니다
✅ 실제 군중이 반응하기 전에, 가상 군중의 합의를 먼저 읽은 것입니다

예측 시장은 결국 사람들의 반응을 예측하는 시장입니다. MiroFish는 가격이 아니라 군중 심리를 시뮬레이션함으로써 이 구조를 정확히 파고들었습니다.

Polymarket도 대응에 나섰습니다. Palantir와 TWG AI를 도입해 스포츠 베팅 감시 인프라를 구축하기 시작했습니다.

AI 에이전트가 시장에 직접 참여하고, 플랫폼이 AI로 감시하는 구조. 이 공방은 이제 시작입니다.
